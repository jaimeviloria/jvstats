# getstats, a commandline tool for retrieving stats

## Requirements
* make
* python3
* virtualenv

## Installation

`sudo make install`
Installs getstats

## Quickstart using Virtualenv

1. `make test`
This creates virtualenv in the Makefile directory

2. `source virtualenv/bin/activate`
